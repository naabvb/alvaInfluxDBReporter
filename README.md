# AlvaInfluxDBReporter

Small utility that fetches electricity consumption data from Alva and uploads it to InfluxDB 1.8.

## Prerequisites

```
Node.js
```

## Installation

```
npm run install
```

## Required environment variables

```
- INFLUXDB_HOST
- INFLUXDB_PORT
- INFLUXDB_PROTOCOL
- INFLUXDB_USERNAME
- INFLUXDB_PASSWORD
- INFLUXDB_DATABASE
- ALVA_USERNAME
- ALVA_PASSWORD
- ALVA_METERING_POINT_CODE
- TZ
```
