# OomiInfluxDBReporter

Small utility that fetches electricity consumption data from Oomi and uploads it to InfluxDB 1.8.

## Prerequisites

```
Node.js
```

## Installation

```
npm run install
```

## Required environment variables

```
- INFLUXDB_HOST
- INFLUXDB_PORT
- INFLUXDB_PROTOCOL
- INFLUXDB_USERNAME
- INFLUXDB_PASSWORD
- INFLUXDB_DATABASE
- OOMI_USERNAME
- OOMI_PASSWORD
- OOMI_CUSTOMER_CODE
- OOMI_NETWORK_CODE
- OOMI_METERING_POINT_CODE
```
